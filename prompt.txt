Build me a premium personal portfolio website for a senior full stack software engineer.

This is not a normal portfolio. It should work both as a recruiter-facing portfolio and as a personal biography site. It must showcase professional work, technical depth, mentorship, leadership, public presence, travel, books, hobbies, and selected life moments in a clean, elegant, high-credibility way.

Use this stack:
- Astro
- TypeScript
- Tailwind CSS v4
- React only where interactivity is needed
- Motion for animation
- Cloudflare Pages for deployment
- Cloudflare Pages Functions for the contact form endpoint
- Cloudflare Turnstile for spam protection
- Resend for email sending, structured so Postmark can be swapped in later if needed

Architecture goals:
- Mostly static site
- No VPS
- Content-driven
- Excellent SEO
- Fast load times
- Minimal JavaScript by default
- Rich interactions only where they add real value
- Clean file structure
- JSON as the main content source of truth
- Strong component reuse
- Production-ready code only
