# Section Components

Card components used in listing pages and content displays.

## ProjectCard

**File:** `src/components/sections/ProjectCard.astro`

A card for displaying a project in listing contexts. Shows the project title, summary, and categories.

## PressCard

**File:** `src/components/sections/PressCard.astro`

A card for displaying a press entry. Shows the title, publication, type badge, summary, and action links.

## WritingCard

**File:** `src/components/sections/WritingCard.astro`

A card for displaying a writing entry. Shows the title, format, summary, and topics.

## WorkFilter

**File:** `src/components/work/WorkFilter.tsx`

A React component for filtering projects by category. Currently not used in the UI (category filtering was removed from the work page) but the component remains available.

## ContactForm

**File:** `src/components/contact/ContactForm.tsx`

The contact form React component. See [Contact Page](../pages/contact.md) for details on its behavior and [Contact Form Overview](../contact-form/overview.md) for the full system.
